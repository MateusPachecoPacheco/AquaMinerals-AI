## 📖 Sobre o Projeto

**AquaMinerals** é uma plataforma científica inovadora dedicada ao estudo e monitoramento da extração sustentável de minerais presentes na água do mar, utilizando **Madre de Deus (BA)** como laboratório natural de referência.

A plataforma integra **ciência, tecnologia e comunidade** através de:
- 🤖 **AquaAI** — Assistente de inteligência artificial especializada em oceanografia
- 📊 **Dashboard em tempo real** — Métricas de salinidade, pH e concentração mineral
- 🗺️ **Mapa interativo geolocalizado** — Pontos de coleta na Baía de Todos-os Santos
- 🌍 **Comunidade ativa** — Denúncias ambientais, sugestões e projetos educativos
- 🔐 **Autenticação segura** — JWT, RBAC (controle de acesso por função)
- 🌓 **Dark/Light mode** — Interface moderna e acessível

## 🚀 Tecnologias

### Frontend
- ⚛️ **React 19** + **Vite 8**
- 🎨 **Tailwind CSS v4** + **shadcn/ui**
- 🛣️ **TanStack Router** (file-based routing)
- 📊 **Recharts** (gráficos e visualizações)
- 🗺️ **Leaflet** (mapas interativos)

### Backend
- 🚀 **Fastify 5** (alta performance)
- 📘 **TypeScript 5** (strict mode)
- 🗄️ **Prisma ORM** + **PostgreSQL 16**
- 🐳 **Docker** (ambiente isolado)
- 🔐 **JWT** + **bcrypt** + **Zod** (validação)
